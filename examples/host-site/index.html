This is a live demonstration of the Kuzushi widget platform. The widget below is a fully-functional AI assistant embedded using a single HTML element.

Features

  • Real-time WebSocket chat
  • OpenAI GPT-4 powered responses
  • Streaming message support
  • WebRTC voice calling (experimental)
  • Shadow DOM isolation
  • Multi-tenant architecture
  • Backend-driven configuration
  • Customizable theming

How to Embed

Add this to any website:

<script src="/widget-loader.js"></script>
<kuzushi-widget
  project-id="demo-support-widget"
  api-base-url="http://localhost:3001/api"
></kuzushi-widget>

Project ID: demo-support-widget
Backend: http://localhost:3001
